# 簡介
以 **ASP.NET Core MVC** 為後端架構製作的簡易購物網站，並使用**SQLite**資料庫來儲存資料。
# 使用流程
- **購買流程**:找到喜歡的商品，點選立刻購買/加入購物車按鈕，兩者的差別:
  - 立刻購買:將商品加入購物車後，會跳轉到購物車頁面。
  - 加入購物車:只將商品加入購物車不跳轉，可以繼續選購商品。<br>
  - :warning:注意:當購物車已存在商品，兩者都會跳出警示。<br>
- **新增商品**:進入新增商品頁面，填好資料送出。
- **修改商品**:進入修改商品頁面，找到要修改的商品，點選修改商品按鈕，修改資料完成後送出。
- **刪除商品**:進入刪除商品頁面，找到要刪除的商品，點選刪除商品按鈕，會讓你再確認一次，確認無誤點選確認。
- **使用者註冊**:進入登入頁面，填好要註冊的帳號密碼，點選註冊按鈕。
# 資料欄位限制
- **商品名稱**:不能空白、不能重複。
- **商品描述**:可以空白。
- **商品價格**:不能為負(Ex:-10)、不能0開頭(Ex:012)。
- **商品圖片**:可以不上傳檔案。
- **帳號**:不能空白、不能重複(註冊時)、不能輸入中文或全形符號。
- **密碼**:不能空白。
# 使用者權限
**未登入**:只能購買商品。<br>
**已登入**:依權限可分為:<br>
:small_orange_diamond:**user**:可以新增、修改、購買商品，有購物紀錄，但**不能刪除商品**。<br>
:small_orange_diamond:**root**:全部都可以，包含刪除商品。<br>
:bangbang:特別注意:使用者只能修改自己新增的商品。Ex:商品是user1新增的，只有user1能修改，其餘使用者不行，即使他的權限為root。
# 預先建立的使用者

|user|user1|123|

# RWD
所有頁面皆採用RWD設計確保版面保持美觀，**寬度**最小可到**300px**，**高度**最小可到**300px**。當**寬度<500px**或**高度<601px**登入頁面的動畫會取消。
